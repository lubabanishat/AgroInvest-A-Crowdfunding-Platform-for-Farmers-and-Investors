# AgroInvest

AgroInvest is an agriculture-based crowdfunding platform that connects verified farmers with small investors.

## Project Objective

The objective of AgroInvest is to help farmers raise funds for agricultural projects and allow investors to invest small amounts and earn a share of the project profit.

## Main Actors

- Farmer
- Investor
- Admin
- SSLCommerz Payment Gateway

## Main Features

- Farmer registration and document verification
- Agricultural project creation
- Admin approval or rejection
- Browse verified farming projects
- Small-scale investment
- SSLCommerz payment integration
- Funding and milestone tracking
- Profit-sharing report

## Project Structure

- `Documentation/` – Proposal, SRS, survey and diagrams
- `UI-UX/` – Figma design, screenshots and prototype
- `frontend/` – Frontend source code
- `backend/` – Backend and REST API
- `database/` – Database design and SQL files
